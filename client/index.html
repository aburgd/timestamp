Timestamp Microservice

What is this?

Takes a string parameter in the URL and returns unix and human-readable values, or else null.

Usage:


      `https://hidden-crag-78313.herokuapp.com/January 10, 2015`
      `https://hidden-crag-78313.herokuapp.com/1420848000`

Example Response:


      {
        humanReadable: "January 10, 2015",
        unix: "1420848000"
      }
